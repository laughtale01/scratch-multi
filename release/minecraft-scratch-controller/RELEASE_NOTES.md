# MinecraftEdu Scratch Controller - リリースノート

## Version 1.0.0 (2025-11-14)

🎉 **本番リリース第1版**

このバージョンは、Scratch 3.0とMinecraft Java Edition 1.20.1を連携させる教育向けプログラミングツールの最初の安定版です。

---

## 新機能

### ✨ コア機能
- **Scratch-Minecraft連携**: WebSocketによるリアルタイム通信
- **Minecraft拡張機能**: Scratch 3.0にMinecraft制御ブロックを追加
- **18種類のブロックタイプ**: 石、土、ガラス、ダイヤモンド等に対応
- **12種類のエンティティ**: ブタ、ウシ、ゾンビ、村人等を召喚可能

### 🎮 ブロック操作
- **絶対座標指定**: x, y, z座標でブロックを配置
- **相対座標指定**: プレイヤー位置からの相対位置に配置（~記法）
- **範囲設置機能**: 最大200,000ブロックを一度に設置
- **Y座標自動変換**: Scratchの座標系からMinecraftの座標系へ自動変換

### 🌍 ワールド操作
- **天気変更**: 晴れ・雨・雷雨の3種類
- **時刻変更**: 朝・昼・夕方・夜・真夜中の5種類
- **プレイヤーテレポート**: 任意の座標への瞬間移動
- **座標取得**: プレイヤーの現在位置（x/y/z）を取得

### 🎨 日本語対応
- 全ブロック名を日本語化
- 全エンティティ名を日本語化
- Minecraftテーマカラー（緑色）でUIを統一

## 技術仕様

### Minecraft MOD
- **対応バージョン**: Minecraft 1.20.1
- **必要環境**: Minecraft Forge 1.20.1
- **言語**: Java 17
- **通信**: WebSocket (ポート14711)
- **依存ライブラリ**: Gson, Java-WebSocket

### Scratch拡張機能
- **Scratch VM**: 5.0.300
- **Scratch GUI**: 5.2.14
- **言語**: JavaScript (ES6+)
- **ビルドツール**: Webpack 5, Babel
- **通信**: WebSocket Client (ブラウザAPI)

---

## 既知の制限事項

### ブロック設置
- 範囲設置は最大200,000ブロックまで（パフォーマンス保護）
- 一部の特殊ブロック（レッドストーン回路、コマンドブロック等）は未対応

### 通信
- ローカルネットワーク内での使用を想定（インターネット越しは未サポート）
- 同時接続は1クライアントのみ

### 互換性
- Minecraft Java Edition専用（統合版/Bedrock Editionは非対応）
- Forge以外のMODローダー（Fabric等）は非対応

---

## パフォーマンス

### ベンチマーク結果（テスト環境: Core i5, 16GB RAM）
- **ブロック設置速度**: 約10,000ブロック/秒
- **WebSocket遅延**: 平均 < 10ms
- **メモリ使用量**: Minecraft +50MB, ブラウザ +100MB

## セキュリティ

### 注意事項
- WebSocketサーバーは認証なし（ローカル使用を想定）
- ポート14711をインターネットに公開しないでください
- サーバー環境での使用は自己責任で行ってください
